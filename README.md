# Gnosys

An attempt at creating a "Language Model build system" (if such a thing makes sense), for educational purposes.

Most of the work and source code is inspired from the following book: `Rashka, Sebastian. 2025. Build a Large Language Model. Manning. ISBN: 9781633437166`
